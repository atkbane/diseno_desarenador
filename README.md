# Diseño de Desarenadores tipo intermitente

## Descripcion
Hoja en jupyter para el calculo de las dimensiones de un desarenador
partiendo de las dimensiones de las particulas que se desean decantar
y el caudal de diseño del proyecto

## Requisitos
- tener instalado visual studio code o anaconda
- instalar los modulos necesarios. (luego seran especificados)
